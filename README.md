# 📈 IPO Watch - 日本IPO銘柄情報データベース

## 概要

**IPO Watch** は、日本の新規上場（IPO）企業の情報を一元管理し、投資判断をサポートするWebアプリケーションです。近日上場予定企業から過去6年間の履歴データまで、包括的なIPO情報を提供します。

---

## 🌟 主な機能

### 📌 近日上場企業情報
- **8社の上場予定企業** を掲載
- 各企業の詳細情報を確認可能
  - 会社概要（設立年、市場区分、事業内容）
  - 公式ウェブサイトへのダイレクトリンク
  - 主幹事証券、取扱証券会社情報

### 📊 初値予想・スコア
- **5つの視点からの企業評価** （レーダーチャート表示）
  - 成長性 / 収益性 / 市場規模 / ブランド力 / テック度
- **初値予想** 範囲を自動計算
- **勝率** と **期待利益率** を表示

### ⭐ マイウォッチリスト
- 気になる銘柄を即座に登録・管理
- ウォッチリスト一覧ページで統計情報も表示
  - 登録銘柄数
  - 平均期待利益率
  - 市場別分布

### 🔔 リマインダー機能
- 上場7日前に自動通知（ブラウザプッシュ通知対応）

### 🌐 SNS シェア
- Twitter / Facebook / LINE で情報をシェア可能

### 📈 過去6年間のIPO履歴
- **471社** の過去IPO企業データ
- 各企業の実績情報（公開価格、初値、収益率）
- 統計分析：平均利益率、勝率、最高利益など

## 💡 使い方

### 1. 近日上場企業をチェック
- トップページの「🔔 近日上場」タブで、これから上場する企業を確認
- 気になる企業名をクリックして、詳細情報（モーダル）を表示

### 2. 企業情報へアクセス
- 各企業の公式ウェブサイトへダイレクトリンク
- 企業の事業内容や投資情報を確認

### 3. ウォッチリストに登録
- 各カードの **★ ボタン** をクリック
- 気になる銘柄を好きなだけ登録

### 4. マイウォッチリストを確認
- ヘッダーの **「⭐ マイウォッチリスト」** ボタンをクリック
- 登録した全銘柄の一覧と統計情報を表示

### 5. リマインダー設定
- ウォッチリスト登録時に、上場7日前の通知を設定可能

### 6. 過去データを分析
- 「📊 過去6年」タブで、471社の履歴を確認
- 統計情報から市場トレンドを分析

## 📱 対応技術

- **フロントエンド**: HTML5 / CSS3 / JavaScript
- **グラフ**: Chart.js 4.4.1（レーダーチャート）
- **データ保存**: localStorage（ブラウザローカルストレージ）
- **通知**: Web Push Notification API
- **配備**: GitHub Pages
